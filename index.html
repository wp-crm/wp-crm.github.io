How WP-CRM works

After you installed the WP-CRM plugin, you will see all your users in All people tab

You will also see the filter in the sidebar where you can filter users by any attribute

In the same filter, you can Visualize user data. You will see graphics for all attributes from your Data tab which have Dropdown or Checkbox Data Input and filled at least in 2 users profiles.

Send a message to all users which you have or to some particular which you choose. You can choose a template or create a new message.

Create your own forms with custom data fields of your choice. You can have multiple forms located at different places on your website.

Set up different notifications for admins and users which filling your forms. personalize notification by using user attributes, like First Name and so on
